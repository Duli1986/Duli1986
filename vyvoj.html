Vývoj

Pro implementaci enterprise aplikací v jazyce Java existuje standard Java Enterprise Edition také Java EE. Java EE, dříve označovaná jako Java 2 Enterprise Edition nebo J2EE, od roku 2018 vyvíjena pod názvem Jakarta EE, je součást platformy Java určená pro vývoj a provoz podnikových aplikací a informačních systémů. Základem pro platformu Java EE je platforma Java SE, nad ní jsou definovány součásti tvořící Java EE.

Pro vývoj samotných aplikací ve standardu Java EE je zapotřebí využít vývojové prostředí (IDE), které usnadní práci vývojářům.

Netbeans

NetBeans IDE je svobodné, zdarma distribuované integrované vývojové prostředí, které vlastní firma Oracle Corporation, které též sponzoruje jeho vývoj.

IntelliJ IDEA

IntelliJ IDEA je integrované vývojové prostředí Java pro vývoj softwaru. IDE bylo vyvinuto JetBrains (dříve známý jako IntelliJ) a je k dispozici jako Apache 2 Licensed Community Edition a v komerčním vydání. Obě vydání mohou být použity pro komerční vývoj.
