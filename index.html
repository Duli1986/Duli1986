O Wildfly

Původní název tohoto projektu byl JBoss Application Server. Nový název vybrala komunita a oznámen byl na konferenci JUDCon 2013 v Brazílii. Hlavní motivací pro hledání nového jména byla snaha odlišit projekt JBoss AS od produktu JBoss Enterprise Application Platform. WildFly bude upstreamem pro JBoss EAP, podobně jako je Fedora pro Red Hat Enterprise Linux.

Aplikační server je vyvíjen v programovacím jazyce Java. Není to jenom tím, že Java je nezávislá na platformě, ale také tím, že existuje standard pro implementaci enterprise aplikací právě v jazyce Java. Tento standard se nazývá Java Enterprise Edition (JEE). Pokud bychom hledali paralelu se světem operačních systémů, mohli bychom zmínit standard POSIX.
